Bem-vindo ao Lumos Data!

Explore o universo mágico de Harry Potter como nunca antes! Lumos Data é uma aplicação interativa que permite navegar por personagens, feitiços, poções, livros e filmes, utilizando dados da PotterDB API.

Navegue pelas seções abaixo ou use a busca nas páginas de Personagens, Feitiços e Poções para encontrar informações específicas.

Sobre Este Projeto

Esta aplicação foi desenvolvida como parte do Desafio de Desenvolvimento de Software da 2ª edição do programa Zetta Lab 2025. O objetivo foi criar uma aplicação responsiva consumindo uma API pública, demonstrando habilidades em front-end, organização de código e boas práticas de desenvolvimento. Agradeço à Agência Zetta por essa oportunidade!
